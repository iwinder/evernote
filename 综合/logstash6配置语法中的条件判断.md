有时您只想在特定条件下过滤或输出事件。为此，您可以使用条件（conditional）。

Logstash中的条件查看和行为与编程语言中的条件相同。

条件语支持if，else if和else语句并且可以嵌套。

条件语法如下：
```
if EXPRESSION {
  ...
} else if EXPRESSION {
  ...
} else {
  ...
}
```

**比较操作**：

- 相等: ```==```, ```!=```, ```<```, ```>```, ```<=```, ```>=```
- 正则: ```=~```(匹配正则), ```!~```(不匹配正则)
- 包含: ```in```(包含), ```not in```(不包含)

**布尔操作**：
- ```and```(与), ```or```(或), ```nand```(非与), ```xor```(非或)

***一元运算符***：

- ```!```(取反)
- ```()```(复合表达式), ```!()```(对复合表达式结果取反)
