1.原生文件浏览
```
选择项目目录/文件 按 ALT+SHIFT +W ，
会弹出菜单点击 System Explorer
```
2.查找Java类文件
```
Ctrl+Shift +T     
```
  查找工作空间（Workspace）构建路径中的可找到Java类文件，不要为找不到类而痛苦，而且可以使用“*”、“？”等通配符。`
3.查找文件
```
Ctrl+Shift +R
```
包括但不限于Java类
4.查找类、方法和属性的引用
···
Ctrl+Shift +G
···

 查找类、方法和属性的引用。这是一个非常实用的快捷键，例如要修改引用某个方法的代码，可以通过【Ctrl+Shift +G】快捷键迅速定位所有引用此方法的位置。
